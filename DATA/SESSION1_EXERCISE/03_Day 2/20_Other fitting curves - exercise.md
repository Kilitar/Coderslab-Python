# Other fitting curves - exercise

Type: Exercise

1. Open the file that you used in the previous linear correlation assignment. Add a sheet named `polynomial`.
2. For the pairs of variables: `popularity` and `tempo`, generate a scatter plot.
3. Enable the fitting of a polynomial function on the graph (set the degree of the polynomial to **3**). Change the color of the generated curve to contrast with the color of the points.
4. Does the generated polynomial curve represent the data characteristic well? Does increasing the degree of the polynomial improve the fit?
