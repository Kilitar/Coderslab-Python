# Linear regression - exercise

Type: Exercise

1. Download and open the `spotify_regression_exercise.xlsx` file.
2. Create a new tab named `regression_exercise`. Launch the Analysis ToolPak add-in and select a regression model.
3. Set a linear model based on the `energy` and `danceability` variables. Enable an additional option that will show the calculated and actual observations on the graph.
4. Interpret the nature of the variables and read the R2 (*R Square*) value. Will the linear model work in this case?

### Good luck!
