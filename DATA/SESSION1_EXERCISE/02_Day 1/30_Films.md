# Films

Type: Exercise

## Film categories

Using `sakila.film_list`:

1. Write a query to display the films with the genres: `Horror`, `Documentary`, and `Family` with the rating of `P` **or** `NC-17`,
2. Using the results of the previous task and `sakila.film_text`, display the descriptions of these films.

Additionally:

3. Sort `sakila.film_list` by the `Category` key - ascending; `Price` - descending,

4. Sort `sakila.film_list` by key `Rating` - ascending; `Length` - descending.
