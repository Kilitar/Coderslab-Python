# Least

Type: Exercise

## Least

Using `sakila.film_list`, write and perform a query that:

1. returns the lowest value in the columns: `price`, `length`,
2. returns the lowest value in the columns: `price`, `length`, `rating`.

What is the difference between the results from subpoints 1 and 2? Where does it come from?

Hint:
To better illustrate which of the values is going to be considered it is best to list the column data using a `SELECT` command.
