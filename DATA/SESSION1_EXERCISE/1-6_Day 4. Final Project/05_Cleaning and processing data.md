# Cleaning and processing data

Type: Exercise

1. Open the `ufc_fights.xlsx` file, with the results of the previous exercise.
2. Check the `fights` table for duplicates. Remove duplicates (if any).
3. Check the data for accuracy and clean it up if necessary. You need to remember about:
  
  - empty values,
  - unnecessary columns,
  - appropriate data types,
  - occurrence of values that have the same meaning, but are written in different ways or with typos; e.g., "Lightweight Bout" is the same as "UFC Lightweight". Values should be unique.
4. Merge the data from the three tables. Join the information from fight location and fighters' metrics tables with the `fights` table.
5. Add variables that determine:
  
  - fighter's age on the day of the bout (if date of birth is provided)
  - name and surname of the bout winner
  - year of the bout
  - weight and height of the fighter in metric units (kilograms, centimeters)
6. Save the results of your work.

### Good luck!
