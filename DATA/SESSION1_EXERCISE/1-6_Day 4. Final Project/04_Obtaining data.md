# Obtaining data

Type: Exercise

1. Download the `ufc.zip` file.
2. Move the tables from the two `.csv` files to the `ufc_fights.xlsx` file.
3. Transform the data from `ufc_fighters.csv` and `ufc_locations.csv` so that each variable is in a separate column.
4. Save the results of your work in the `ufc_fights.xlsx` file.
