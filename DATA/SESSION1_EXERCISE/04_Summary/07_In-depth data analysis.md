# In-depth data analysis

Type: Exercise

1. Open the `cars_data.xlsx` file used in earlier exercises.
2. Add a new sheet named "in_depth_analysis", where you'll conduct an in-depth exploratory analysis.
3. For the questions where you need to check dependencies:

- calculate Pearson correlation coefficient
- present the dependencies using scatter plots (you can use either the Analysis ToolPak or generate a scatter plot with line estimation)
- assess whether the examined dependencies are linear. Based on the independent variable (i.e., the one on the x-axis), is it possible to determine the value of the dependent variable using a line estimation (linear regression) formula?

1. Save the results of your work.
