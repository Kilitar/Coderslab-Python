## Excel — cheat sheet

Below you'll find a brief summary of the tools we used in the prework, but also those that will be useful in the later stages of the course.

### Keyboard shortcuts

| Shortcut | Action |
| --- | --- |
| F2 | Edit Active cell |
| F5 | Go to chosen cell in the sheet |
| CTRL + arrow | Go to the edge of data in the sheet |
| Shift + F11 | Insert new sheet |
| Alt + = | Sum of cells |
| Ctrl + Shift + “+” | Insert new row/column |
| Ctrl + “-” | Delete row/column |
| Ctrl + “*” | Select all cells with formulas |
| Ctrl + ‘ | Copy value of the cell one row higher |
| Alt + Enter | Add line break inside cell |

### Formulas

| Formula | Sample use | Description |
| --- | --- | --- |
| =SUM(range) | =SUM(A1:A20) | Calculates the sum from the given range |
| =AVERAGE(range) | =AVERAGE(A1:A20) | Calculates the average of a given range |
| =MEDIAN(range) | =MEDIAN(A1:A20) | Calculates the median (middle value) from a given range |
| =PERCENTILE(range; k) | =PERCENTILE(A1:A20; 0.25) | Calculates the percentile k value from the given range |
| =LOWER(text) | =LOWER(A1) | Changes all the characters in the cell to lowercase |
| =UPPER(text) | =UPPER(A1) | Converts all characters to uppercase in the text contained in a cell |
| =LEFT(text; num_char) | =LEFT(A1; 7) | Returns the chosen number of characters in the string counting from the left |
| =RIGHT(text; num_char) | =RIGHT(A1; 6) | Returns the chosen number of characters in the string counting from the right |
| =IF(condition; value_if_true; value_if_false) | =IF(A1=5; "Match"; "No match") | Checks a condition and then returns a value depending on whether the condition is met or not. In the case where we have the number 5 in cell A1, then the formula will return 5. Otherwise it returns 0. |
| =XLOOKUP(lookup_value; lookup_array; return_array) | =XLOOKUP(C2; A1:A10; B1:D10) | The function looks for the value from cell C2 in the range A1:A10 and for the row where it found the value of C2, returns the value of that row located in the B1:B10 range |
| =LEN(text) | =LEN(A1) | Returns the number of characters in the text located in the given cell |
| =CEILING(number; significance) | =CEILING(A1; 0.01) | Rounds the number up within the given significance. For A1 = 2.478 function returns 2.48 |
| =FLOOR(number; significance) | =CEILING(A1; 0.01) | Rounds the number down within the given significance. For A1 = 2.478 function returns 2.47 |

### Charts

| Chart | Use case |
| --- | --- |
| Column | Used to compare neighboring values (columns). |
| Line | Used to show trends in time. Each of the values is a point, and all points are connected with a line. |
| Bar | Similar to column chart, but with horizontal bars instead of columns. |
| Area | Similar to line chart apart from having the area below the line filled with color. |
| Scatter plot | Used to chart clusters of values by using individual points. Many elements can be charted with various colored points (or other symbols). |
| Pie | Popular but not recommended; used to show values as percentages of the whole when they total at 100%. Value of each slice is represented with a different color |
| Candlestick | Effectively reports changes in stock prices such as low and high value as well as the opening/closing prices in the period of time. |
