# Interesting public API

Type: Article

At [https://github.com/public-apis/public-apis](https://github.com/public-apis/public-apis) there is a long list of APIs, that you can connect to with your script in minutes.

In addition to this API have various tools and platforms available online: Jira, Trello, Github, Google (including Google Drive), Facebook, Instagram, Twitter, Youtube....

Some Polish cities, such as: [Warsaw](https://api.um.warszawa.pl/) and [Wrocław](https://www.wroclaw.pl/open-data/) also share their data this way.
