# Useful Python libraries

Type: Article

You already know the `openpyxl` used to work with MS Excel and the `csv` library to work with the files having the same extension. During the course you will learn about many more useful libraries, but we can't show them all. Here are some other libraries you may find interesting:

## Built into Python

- [datetime](https://docs.python.org/3/library/datetime.html) - a collection of functions and classes to represent time: date, hours, time zones, etc.
- [zipfile](https://docs.python.org/3/library/zipfile.html) - everything you need to work with **zip** archives,
- [smtplib](https://docs.python.org/3/library/smtplib.html) - complete support for receiving and sending emails.

## To install with `pip`

- [pillow](https://pillow.readthedocs.io/en/stable/) - image processing and generation,
- [termcolor](https://pypi.org/project/termcolor/) - add colors to `print()` outputs,
- [numpy](https://numpy.org/) - effective processing of large datasets,
- [Scikit Learn](https://scikit-learn.org/stable/) - a library supporting creation of neuron networks and implementation of machine learning algorithms,
- [NLTK (Natural Language Toolkit)](http://www.nltk.org/) - library for analysis of written natural language. Allows classification, tagging, stemming and sentiment analysis. Unfortunately, it only works for English.
- [Fire](https://github.com/google/python-fire) - library supporting command line interfaces creation
- [MoviePy](https://github.com/Zulko/moviepy) - library facilitating video editing with Python.
