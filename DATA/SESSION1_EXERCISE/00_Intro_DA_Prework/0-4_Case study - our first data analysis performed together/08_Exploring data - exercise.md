# Exploring data - exercise

Type: Exercise

In the article we mentioned that your task will be to complete the data exploration process and provide the missing answers to some questions.

Remember that we're looking for answers to the following questions:

- ~~What is the average price for one night rental?~~
- How many of the hosts have more than one listing on the site?
- ~~How does the number of offers break down by type of room?~~
- What is the number of units by neighborhood? For what reason do some neighborhoods offer more rentals?
- ~~In which neighborhoods will the rent be highest?~~
- Are hosts more likely to rent units for short or long periods?

1. Download the file with data and analysis performed so far from the LMS
2. In the "EDA" sheet modify the table so that it includes statistics for the remaining variables – `calculated_host_listings`, `availability_365` and `number_of_reviews_ltm`. After calculating the statistics, interpret their values in a similar way to the one in the article. Look at the source data using filters.
3. Add pivot tables to the "EDA_pivot_tables" sheet to calculate:
  
  - The number of rooms by minimum number of nights per rental,
  - The number of hosts by number of listings, along with what percentage of the total they represent,
  - The number of listings, average price, average number of reviews per month and average availability over the next 365 days by neighborhood (you can modify the existing table).
