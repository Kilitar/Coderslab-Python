# In-depth data analysis - exercise

Type: Exercise

Let's assume we have a hypothesis - there is a linear relationship between the minimum number of nights and the number of reviews (as the minimum number of nights increases, the number of reviews decreases, due to fewer tenants over time).
Your task will be to generate a trend line using a linear model.

1. Download the file with data from the LMS
2. Go to the `in_depth_analysis` sheet; from the "Insert" section select the "Scatter XY" icon and "Scatter" chart.
3. Mark a range in the source data for the variables X (minimum number of nights) and Y (number of opinions).
4. Add a linear trend function to the chart.
5. Add the equation of the generated straight line to the chart, as well as the value of the R2 coefficient.
6. Describe the observed relationship, determine the nature of the function (increasing, decreasing or constant). Is the relationship linear?
