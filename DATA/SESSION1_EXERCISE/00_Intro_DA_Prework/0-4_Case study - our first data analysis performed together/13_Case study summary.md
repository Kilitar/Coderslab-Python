# Case study summary

Type: Article

Congratulations! Together we performed the first data analysis, fully compliant with the guidelines of the `Data Science` process.

In the first step, we focused on understanding the problem and asking the supporting questions, which were:

- aim of the analysis
- the meaning of the various variables in the data we were provided with

We received the data as a single source file, so relatively quickly we were able to **collect** it and start working with the data properly.

We **cleaned and processed** the data we received, taking into account various types of errors, such as blank values or incorrect data format.
We described ways to deal with data errors that were needed in our analysis of listings published on **Airbnb**.

The fourth step was data exploration, and here we looked deeper into individual variables.
We counted and interpreted basic statistics and created pivot tables that aggregated the data to achieve some of the goals we set for this analysis.

We then mentioned what **in-depth data analysis** is and used Excel for a simple analysis of selected pairs of variables, looking for linear relationships in the data. For this, we used a trend line, determined using a **linear model**.

Finally, we discussed the importance of charts in data analysis, and how to choose one depending on what data we need to present.
We collected the created charts and made a PowerPoint presentation using them to summarize the work we had done.
**Communication of results** presented in such a form will certainly make understanding the data easier and quicker.

There is a whole lot of knowledge and challenges ahead of you in the context of data analysis, and the first opportunity to do so will come soon at the first session.

### Good luck!
