# Communicating results - exercise

Type: Exercise

1. Add a column chart that shows the ranking of the most expensive neighborhoods regarding rentals. Limit the number of neighborhoods to **20**.
2. Add a bar chart and present how the number of hosts is distributed according to `minimum_nights`. On the pivot bar chart, add a filter to limit the number of values of `minimum_nights` and increase the readability of the chart.
3. Add the charts as slides in your presentation (each to a separate slide) in the places appropriate to presenting the information in the top-down approach: from general to specific.
4. Move the files with Excel exercise solutions and the file with PowerPoint slides to one folder and compress them to a `.zip` file called `prework_solution.zip`. You can send the file in this format as part of your prework solution to your mentor on Slack.
