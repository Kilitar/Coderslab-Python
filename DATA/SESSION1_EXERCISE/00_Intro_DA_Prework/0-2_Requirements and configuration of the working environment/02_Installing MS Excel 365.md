# Installing MS Excel 365

Type: Article

During the course, Coders Lab guarantees access to Microsoft 365. Login information will be sent individually to each person taking the course.
Once you receive the information, you'll need to log into your account and install Excel. The installation is described [here](https://support.microsoft.com/en-us/office/download-and-install-or-reinstall-microsoft-365-or-office-2021-on-a-pc-or-mac-4414eaaf-0478-48be-9c42-23adc4716658).

Read the article before installation. It contains important information on how to configure the program.

The minimum requirements for your computer are:

- Windows 7 Service Pack 1 or higher
- 1GB RAM (32-bit) / 2GB RAM (64-bit),
- 3GB of free hard drive space,
- screen resolution: 1024px x 768px or higher.

If you're asked about language settings, choose American English or English (United States).

## Verify the language configuration

If you selected language and local settings for US during installation, you should be greeted by a window similar to the following one, when you start Excel:

or a window with a worksheet already open:

![Excel - sheet](/presentations/DTL/en/4.9/W/M_00_S_02/7c63f3d0-4478-47e9-9966-3990a7352210/student_content/0ae1e129-4463-40c8-8b92-9cc2ff36e6e0/images/Excel_2.PNG)

If for any reason your Excel is not in English open the menu corresponding to `File` and go to `Options` in the lower-left corner of the screen:

![Excel - changing language](/presentations/DTL/en/4.9/W/M_00_S_02/7c63f3d0-4478-47e9-9966-3990a7352210/student_content/0ae1e129-4463-40c8-8b92-9cc2ff36e6e0/images/Excel_3.PNG)

Then go to language settings and set English as the display and creation language. When a dialog box appears asking if you want to set this language as the default - confirm. The language pack will be installed and you may need to restart Office.

This is how the `Options` menu looks like after the language is switched to English:

![Excel - changing language in options](/presentations/DTL/en/4.9/W/M_00_S_02/7c63f3d0-4478-47e9-9966-3990a7352210/student_content/0ae1e129-4463-40c8-8b92-9cc2ff36e6e0/images/Excel_4.PNG)

By default, Excel uses decimal and thousandth place separators according to your Windows user settings. This is sometimes a problem when working in an international environment. You can change these settings in the advanced options menu:

![Excel - separators](/presentations/DTL/en/4.9/W/M_00_S_02/7c63f3d0-4478-47e9-9966-3990a7352210/student_content/0ae1e129-4463-40c8-8b92-9cc2ff36e6e0/images/Excel_5.PNG)

This is a fairly common problem and it is difficult to avoid. Therefore, in the course, you'll learn how to deal with discrepancies in data saving formats occurring due to the local settings used when saving the file.
