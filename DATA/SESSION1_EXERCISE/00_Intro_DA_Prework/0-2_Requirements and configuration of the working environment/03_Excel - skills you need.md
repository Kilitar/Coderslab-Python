# Excel - skills you need

Type: Article

To effectively go through the course, you need to already have knowledge and skills covering certain areas of working with Excel.

Below you'll find a description of Excel skills that we believe can be considered a good level of Excel knowledge. If you don't have good Excel foundations, such as editing data, creating formulas, charts and pivot tables, it will be difficult for you to complete and acquire the course material and focus on more advanced data analysis topics.

We expect that you already have knowledge and skills covering the following areas of working with Excel:

## Working with data

- entering and editing data in cells and the formula bar,
- formatting data (data types in Excel),
- formatting cells (fonts, fill colors, etc.),
- **types of referencing in Excel** (addressing cells: relative, **absolute** e.g.: '$A$1' and mixed),
- working with multiple sheets of data,
- conditional formatting,
- table format.

## Formulas in Excel

- Basic knowledge of formula syntax and formula manipulation:
  
  
  
  - **required and non-required arguments**,
  - **dragging formulas**,
  - **working with data ranges** in formulas; absolute and mixed addressing,
  - **nesting formulas**.
- Knowing functions:
  
  
  
  - arithmetical: **SUM(), SUBTOTAL()**
  - statistical: **AVERAGE(), MAX(), MIN(), COUNTBLANK(), COUNTA()**
  - date and time, e.g.: **DATE(), YEAR(), MONTH(), WEEKNUM(), DAY()**
  - logical: **IF()**
  - searching and linking: **VLOOKUP(), HLOOKUP()**
  - text: **'&', CONCATENATE()**

## Filtering and sorting

- ability to **insert filters** to data tables,
- applying complex value filters,
- **sorting by one and multiple columns**,
- applying SUBTOTAL() formulas in conjunction with filters.

## Charts

- The **skill of creating charts**:
  
  - ability to **prepare data for visualization** in Excel,
  - inserting charts: line, bar, and point charts,
  - editing charts and formatting their elements:
    
    - chart title,
    - axes (units and value ranges) and axis titles,
    - auxiliary axes,
    - data series,
    - legend,
    - data labels,
    - coloring styles,
  - inserting statistical charts: histogram,
    
    - billing.

## Pivot tables

- **The ability to create and work with pivot tables**:
  
  - adding **row variables that define aggregation ranges**,
  - adding multiple variables for aggregation and editing aggregation functions,
  - adding **columns specifying aggregation ranges**,
  - ability to work with **dates** in pivot tables,
  - grouping data in pivot tables,
  - creating and applying filters to pivot charts,
  - referencing data in pivot tables,
- **Pivot charts**:
  
  - creating pivot charts,
  - ciltering data in pivot charts.

## Practical functions in Excel

- **Text to columns** and its advanced options,
- **Remove duplicates**,
- find and replace function,
- locking rows and columns,
- data protection in Excel:
  
  - blocking contents,
  - structure protection,
  - file encryption and password protection,
- comments and notes.

### Good luck!
