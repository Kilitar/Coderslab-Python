# Excel — review exercises

Before we go any further, we want to test your Excel knowledge with a practical task. We attached a dataset from [Kaggle](https://www.kaggle.com/datasets/micheldc55/anual-salary-reports-survey), with data about salaries of a polled group of people.

- `ID` — interviewee id
- `income` — annual income in dollars
- `age` — interviewee age
- `gender` — interviewee gender
- `education_level` — where 0 — primary, 1 — secondary, 2 — tertiary, 3 — postgraduate

1. In the source file correct the data in `income` and `education_level` columns so that it is possible to perform mathematical operations on them.
2. Remove rows from the worksheet that have blank data in any of the columns.
3. Calculate the average, median, maximum value and minimum value for age and annual earnings.
4. Turn on filters Set them to show data only for people who are older than 25, earn more than 50000 per year and have a minimum of secondary education. Determine how many people (ID) there are who meet such criteria.
5. Clean up the filtering and create a pivot table to determine how much women and men earn on average according to their education level.
  Use the prepared data to create a bar chart.
