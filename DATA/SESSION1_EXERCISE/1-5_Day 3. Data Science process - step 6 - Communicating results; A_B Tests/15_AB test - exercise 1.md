# AB test - exercise 1

Type: Exercise

Spotify uses a recommendation algorithm that suggests songs that users might be interested in listening to. We decided that we wanted to improve our algorithm a bit so that users would receive music content that is a little more tailored to their preferences.

1. Download and open the `ab_test_exercises.xlsx` file.
2. In the `ab_test_1` tab you'll find the results of the A/B, that:
  
  1. in **version A** of the application we tested the basic recommendation algorithm, and in **version B** we tested the algorithm with slightly changed weights,
  2. the purpose of the test is to determine whether the algorithm in version B is better than the one in version A (i.e., whether it resulted in an increase in the number of plays relative to the number of recommended songs),
  3. lasted 14 days,
  4. level of statistical significance was set at **0.05**.
3. Perform an aggregation of the test results and record the results in a separate table.
4. Using the [AB+ Test by CXL](https://cxl.com/ab-test-calculator/), calculator decide whether the test is statistically significant and yielded an increase, and how the introduced change could impact the recommendation effect after it is implemented in the service.
  If there is no statistical significance, consider what could be done to make the test achieve it.

### Good luck!
