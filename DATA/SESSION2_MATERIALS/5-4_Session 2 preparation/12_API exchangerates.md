# API exchangerates

Type: Article

[At this address](https://exchangeratesapi.io/) you will find a website that uses API to provide currency exchange rates. In the upper right corner of the page, click "GET FREE API KEY", then select the "Free" plan and complete the registration. Once registered, the site will generate a key (token) for you to use the API. Copy this key and save it in a safe place - you will need it to work with the site's API.

This API documentation is at  [https://exchangeratesapi.io/documentation/](https://exchangeratesapi.io/documentation/).

**Exercise 2** will help you confirm that your key works.
