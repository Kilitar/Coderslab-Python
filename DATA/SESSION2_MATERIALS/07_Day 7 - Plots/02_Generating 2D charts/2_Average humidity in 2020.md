# Average humidity in 2020

Type: Exercise

Based on the data from file `Meteo data.csv`, create a plot showing the average daily relative humidity in Warsaw in 2020.

The plot should meet the following requirements:

1. The x-axis should be labeled with months.
2. Both x-axis and y-axis should have appropriate descriptions.
3. The plot should include a title.
4. The plot should include a legend.
5. The line of the plot should be red, continuous.
