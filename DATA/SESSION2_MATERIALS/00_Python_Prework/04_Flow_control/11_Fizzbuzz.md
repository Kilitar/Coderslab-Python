# Fizzbuzz

Type: Exercise

For numbers from 1 to 100 (inclusive), write a program that:

- if the number is divisible by 3, output `Fizz` (for example, for the number 6, `Fizz` should be displayed),
- if the number is divisible by 5, output `Buzz` (for example, for the number 10, `Buzz` should be displayed),
- if the number is divisible by 3 and 5, output `FizzBuzz` (for example, for the number 15, **only** `FizzBuzz` should be displayed),
- otherwise, output the number on the screen (for example for  4 the output should be `4`).
