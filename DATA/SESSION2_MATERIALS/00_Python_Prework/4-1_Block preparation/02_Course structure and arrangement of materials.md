# Course structure and arrangement of materials

Type: Article

At the very beginning, it is useful to understand what you're going to learn during the course and in what order.

The course consists of a **prework** and **5 2-day sessions**. Each session consists of 3 parts:

1. preparation materials,
2. materials for classes with the lecturer,
3. summary materials.

### 1. Preparation materials

Before each session, you're going to find a package of materials in the LMS that will prepare you for the class work. These materials will consist of articles, quizzes and exercises. You will also find solutions to all the exercises that come up during your preparation. This way you won't get blocked at the stage of preparing for the class. However, we recommend that you reach for the solutions only if you are stuck - or after you solved the task yourself.

In the preparation materials you will also find all the instructions that prepare your computer for work during the class.

Conscientiously going through the preparatory materials is a requirement. Failure to familiarize yourself with these topics can result in problems with understanding the subject matter or technical problems during class. This can make the lecturer's work very difficult and disrupt the group's work - so be sure to report any problems that arise while preparing for the session to your lecturer before class begins.

### 2. Materials for classes with the lecturer

For each session there will be materials to go through with the lecturer. These materials are going to consist of presentations, exercises, and additional files that will help you understand the topic or will be needed to do the exercises. The materials are divided for each day, so that it is easier to keep the files organized for a particular day. There are no solutions available in the LMS for the exercises planned for the class with the lecturer.

Remember to work diligently during classes with the lecturer: to rewrite the examples that the lecturer prepared and execute them, to solve the tasks in the time allocated for this. Also, don't be afraid to ask questions - during this course you will encounter many topics that will be a complete novelty, the lecturers are aware of this. You can always ask the lecturer to spend a moment with you in a 1-on-1 conversation.

### 3. Summary materials

After each session there will be summary materials in the LMS. They consist of articles, quizzes and exercises with solutions. These materials are intended to summarize the knowledge from a given session, but also to show further opportunities to develop the new knowledge. The exercises are cross-sectional: they combine the new knowledge with what was discussed earlier.

## What is prework?

Before everything begins, the first part of the course is prework. These are materials to be done on your own, provided in the LMS. You work on them remotely and independently - without the lecturer. This part of the course is designed to prepare you to attend classes with the lecturer.

Prework consists of the following subjects:

- course introduction,
- computer configuration,
- Python programming basics,
- SQL database basics.

You have to complete the prework before attending the first class with the lecturer. Passing requires sending all exercise solutions (both Python and SQL) to the mentor assigned to the group.

**Comprehensive** completion of the prework is very important. The knowledge in it will be needed from the first day of the course.

### How to do the prework?

Do the prework in the order of the materials in the LMS:

1. Familiarize yourself with the rules of the course - if something is not clear, go ahead and write to the course specialist.
2. Install all the necessary programs and configure your computer. If you encounter any problems during this step - reach out to the course mentor. They will certainly help with the configuration and installation.
3. Do the materials related to learning Python language. The knowledge from this part of the prework will be developed further from the very first class. Therefore, understanding and mastering these topics is fundamental.
4. Finally, review the materials on SQL. This knowledge will be needed during the second session.

## Course modules

As we mentioned earlier, the course consists of 5 sessions. Each session is 2 days of work with the trainer, study time from 9am to 5pm. The sessions cover the following topics:

### Session I — Python basics

Python programming techniques will be used from the first day of the course. During the first 2 days you're going to learn how to use the most popular data structures such as lists, dictionaries and tuples. Then you will learn how to write functions and why code partitioning is so important. You will learn working with strings and the basics of using regular expressions. The session ends with an introduction to object-oriented programming and the use of libraries in Python - so you'll get the benefit of ready-made solutions created by hundreds of programmers around the world.

### Session II — SQL and API

Databases are perhaps the most popular source of data for analysts. During this session, you'll learn how to use them with SQL database queries. You'll learn how to connect to a database using Python. Then you will find out how to join tables in a database using SQL queries. You'll also discuss the most important analytical functions available in SQL.
The second day focuses on retrieving data from applications using their API. This will be an easy way for you to quickly retrieve data from a wide variety of applications. The JSON file format - the most popular format used when communicating with APIs - will also be discussed.

### Session III — Pandas

The entire fourth session will be devoted to the most important tool presented in the course – Pandas. This is a library used for data analysis and for quick work with data. During this session you are going to learn what series and data sets are, and how they can be used to analyze data. Using examples, you will see how data can be joined together and how to clean it of unnecessary information.

### Session IV — WebScraping and plotting charts

For an analyst's report to be complete, it cannot lack charts. There are many types of charts, and during this module we will discuss creating them using Python. This will make it very easy to generate new charts for data that changes.
On the second day, during the web scraping class, you will learn how to obtain data from websites that do not provide an external API for developers. You'll get a basic understanding of HTML code and learn how to search for elements of interest in it. You will also write the first program allowing you to automatically retrieve data from a website.

### Session V — Plotting charts and workshop

The last topic introduced covers generating a PDF using Python code. This will help you automate a lot of manual work.
The last day is dedicated to the workshop. During the workshop you are going to create a project to analyze jobs in Poland. You'll create a program that takes data from several sources, combines them, and then generates a report from them. An additional development will be saving the data to a database - to make future use easier.
