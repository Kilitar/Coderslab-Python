# Retrieving data from the user

Type: Exercise

Write a program, that:

1. Assigns user's first name to a variable (retrieving it with an appropriate function),
2. Assigns user's surname to a variable (retrieving it with an appropriate function),
3. display a message: `"Name Surname is a Python programmer!"`.

Example of calling the code

``` 
What's your name: Guido
What's your surname: van Rossum
Guido van Rossum is a Python programmer!
```
