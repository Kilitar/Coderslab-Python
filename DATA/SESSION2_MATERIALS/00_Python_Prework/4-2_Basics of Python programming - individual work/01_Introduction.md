## Introduction to Python — self-study

Welcome to the part of the course dedicated to learning the Python programming language. In the following articles you will learn the basics of this language, which you are going to need later, when creating scripts.

It is very important to go through this part of the prework - you will use the knowledge contained here from the very first classes with the lecturer.

### What will you learn?

After going through this chapter you'll have the following knowledge:

- You'll know the basic terms used by programmers, such as: code, program, pseudo-code etc.
- You'll be able to run a Python program: both in Jupiter and in the interactive Python shell.
- You'll learn the basic data types in Python and how to use them.
- You'll be able to control a program using loops and conditional expressions.
- You'll learn how to write programs in Python in such a way that they are also readable for other programmers.
  Your knowledge of each of these points is going to be tested with a series of exercises to solve. They are prepared in such a way that any person, even without previous programming experience, will be able to solve them.

## How to do the exercises?

To make completing exercises easier we prepared the file `Basics of Python programming - individual work.zip`, than contains all the files necessary to complete the exercises. To get started, all you need to do is download this file from the LMS platform (it's the next item in this chapter), unzip it, and open the given folder using Jupyter Notebook.

For each exercise, a Jupyter file has been prepared with the content of the task (the same as on the LMS), and a place to enter the solution.

Remember to send the set of solutions, **zipped in the same way** to your mentor as your credit work for this part of the prework. **A passing score of 80% on the prework is required to join the lecturer's classes!
