# Multiplication table

Type: Exercise

Write a program, that:

1. Takes an `n` number in the range from 1 to 10 from the user.
2. Generates a multiplication table for the given number.

**Example**:

``` 
Give a number: `3`
```

**Result**:

``` 
1 * 3 = 3
2 * 3 = 6
3 * 3 = 9
4 * 3 = 12
5 * 3 = 15
6 * 3 = 18
7 * 3 = 21
8 * 3 = 24
9 * 3 = 27
10 * 3 = 30
```

> **Hint:**
> 
> Remember to convert `n`.
