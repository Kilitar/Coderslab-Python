# Defining list of numbers

Type: Exercise

1. Define a list consisting of **even** numbers from `2 to 98`,
2. List each of these numbers on a separate line using the template below:

``` 
Number: <number>
```

> **Hint:**
> 
> Try not to define the list manually; uste the `range` function instead.

Result draft:

``` 
Number: 2
Number: 4
...
Number: 98
```
