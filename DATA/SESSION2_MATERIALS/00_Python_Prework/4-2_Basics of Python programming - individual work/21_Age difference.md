# Age difference

Type: Exercise

1. Create a variable called `father` and assign the value of `1974` to it.
2. Create a second variable named `child` and give it the value `2007`.
3. Print a message formatted as: `"The father is <difference> years older than the child."`.

> **Hint:**
> 
> It is most convenient to use the `fstring` construct or the `format` method.
