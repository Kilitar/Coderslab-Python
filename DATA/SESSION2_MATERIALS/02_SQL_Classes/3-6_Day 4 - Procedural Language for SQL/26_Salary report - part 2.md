# Salary report - part 2

Type: Exercise

## Salary report - part 2

Extend the query from the previous task and check for differences in earnings in the following groups:

- divided by gender, and department,
- divided by gender,
- no division.

Display the data in a way that is easy to read.

> Remember to add information on how many people were in each group. On this basis it will be possible to determine whether the groups are comparable.
