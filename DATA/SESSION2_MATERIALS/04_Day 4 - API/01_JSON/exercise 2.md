# Exercise 2

Choose all customers from the `classicmodels` database. Iterate over them and use the [openweathermap.org](https://openweathermap.org/) API to check their weather:

- if it is rainy (`"Rain"` in the appropriate part of the API response), display the customer name as well as name and surname and telephone number of the contact person,
- in all other cases display only the customer name.

Use the API key from the introduction to the second session.
