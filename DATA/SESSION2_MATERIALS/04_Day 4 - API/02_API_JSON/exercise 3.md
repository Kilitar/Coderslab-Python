# Exercise 3

Using the `requests` library retrieve the current exchange rates from [https://exchangeratesapi.io/](https://exchangeratesapi.io/) and display them as a list:

``` 
- <RATE 1> EUR/<CURRENCY 1>
- <RATE 2> EUR/<CURRENCY 2>
...
```

The result should be similar to:

``` 
- 4.5066 EUR/PLN
- 1.5319 EUR/CAD
- 0.86308 EUR/GBP
...
```
