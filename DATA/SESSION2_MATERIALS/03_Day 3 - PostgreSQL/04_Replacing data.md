# Replacing data

Type: Exercise

## Replacing data

Write SQL queries that:

1. Change the location of an employee: `Bott Larry` from the office with id `7` to the one with id `5`.
2. They will change the credit limit for the company 'Tekni Collectables Inc.' to 60,000.
3. Change the number of available items of all products from the `Trucks and Buses` product line to 0.
4. Change the number of ordered items with code `S18_2957` in order number `10103` to 60.
