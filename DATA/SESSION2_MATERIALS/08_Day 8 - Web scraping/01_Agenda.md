# Agenda

Type: Article

## What will you learn today?

1. You will learn how data can be extracted from web pages.
2. You'll learn the basics of HTML.
3. You'll get to know the `BeautifulSoup` library for effective work with HTML.
4. You'll combine the `BeautifulSoup` library with the previously learned `requests` library.
5. You'll learn how to control a browser using Python.
