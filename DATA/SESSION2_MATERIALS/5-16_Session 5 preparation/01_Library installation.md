# Library installation

Type: Article

We will use the `reportlab` library to generate PDF files. It allows you to create PDF files in several ways. In this class we will discuss two of them:

- manually specifying where on the page a given element (e.g. text) should go,
- Story, i.e. adding more content in the order it appears on the page.

You can find the documentation [here](https://www.reportlab.com/docs/reportlab-userguide.pdf).

You can install the library using any of the ways you know: `pip` or `conda` commands or with Anaconda Navigator GUI.
